Pantallas para Renfe

Home Trabajos

Diseño de pantallas táctiles con sistema de entretenimiento, para viajes de larga distancia.

ROL: UX Designer único.

CONTEXTO: Sistema de entretenimiento para Renfe

HERRAMIENTAS: Sketch, Principle

Estas pantallas están diseñada, para el mayor placer de los usuarios. Con este nuevo software los viajeros, tienen todas las opciones para que su viaje sea los mas placentero del mundo, ya que tienen todo tipo de variantes para divertirse, desde escuchar todo tipo de música a ver películas.

También tiene una parte de restauración, donde tienen una carta muy variada, pero si en algún caso tuvieran una duda o alguna consulta, podrán llamar a los azafatos desde su propio asiento
